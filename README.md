# 🚀 Vibecoding Hackathon

A competitive hackathon where participants recreate existing iOS apps within 24 hours. This repository contains the hackathon platform and starter materials.

## 📖 Overview

The Vibecoding Hackathon is a friendly competition where two participants compete to recreate a selected iOS app in just 24 hours. There are no restrictions on technology stack - choose whatever works best for you!

### Current Iteration: #1

## 🏆 Judging Criteria

Submissions are evaluated on four key areas:

1. **Creativity** - How innovative and unique is your approach?
2. **Design** - How visually appealing and well-designed is the UI?
3. **Similarity** - How closely does it match the original iOS app?
4. **Performance** - How well does it run and function?

Each criterion is scored from 1-10, for a total maximum score of 40 points.

## 🚀 Quick Start for Participants

1. **Clone this repository**
   ```bash
   git clone <repository-url>
   cd menorca-hackthon
   ```

2. **Read the STARTER.md guide**
   - Contains setup instructions and guidelines
   - No template provided - choose your own stack!

3. **Start building**
   - You have 24 hours from the start time
   - Track progress on the hackathon website
   - Submit your public GitHub repo URL when done

4. **Submit your project**
   - Make your repository public
   - Submit your GitHub URL via the submission form on the website
   - Include setup instructions in your README
